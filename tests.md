This page documents test problems implemented in [AGILE](https://github.com/amrvac/AGILE-experimental), a GPU-accelerated MHD code built on the [MPI-AMRVAC](http://amrvac.org) framework.
All tests are run in 3D.

See also:
[MPI-AMRVAC tests](http://amrvac.org/test.html).
[Athena test suite (Stone)](https://www.astro.princeton.edu/~jstone/Athena/tests/),

| Test | Type | Description |
|------|------|-------------|
| [Orszag-Tang vortex](tests/orszag-tang) | MHD | Transition to MHD turbulence |
| [Riemann problems](tests/riemann) | MHD | Collection of 1D shock tube tests |
