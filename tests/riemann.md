## Description

A collection of 1D MHD Riemann problems (shock tubes) used to validate
AGILE's treatment of shocks, rarefactions, and contact discontinuities.

Since AGILE is a fully 3D code, these are run as extruded 1D problems:
the initial conditions vary only along the $x$-direction and the domain is
only a few cells deep in $y$ and $z$. Results are taken along a 1D slice.
